Acerca de RAWG-API

RAWG-API es un proyecto que permite a los usuarios buscar y explorar información sobre videojuegos utilizando la API de RAWG. Este proyecto está desarrollado con Angular y proporciona una interfaz de usuario intuitiva para acceder a una amplia base de datos de juegos.

Características

  • Búsqueda de juegos por nombre
  • Filtrado de juegos por plataforma
  • Visualización de detalles de los juegos

Tecnologías Utilizadas

  • Angular
  • Bootstrap
  • API de RAWG
